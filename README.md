# Weather Wizard CV

A computer vision project that creates interactive weather effects based on hand gestures.

## Features
- Hand gesture recognition using MediaPipe
- Real-time weather particle effects (rain, snow, lightning)
- Interactive visual feedback

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python -m app.main
```

## Gestures
- **1 finger up**: Lightning effect
- **2 fingers up**: Snow effect  
- **3 fingers up**: Rain effect
